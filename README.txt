PWA - Controle Fiscal Day Trade

Arquivos incluídos:
- index.html
- manifest.json
- service-worker.js
- icons/

Publicação:
1. Envie todo o conteúdo da pasta para a raiz do repositório.
2. Ative o GitHub Pages.
3. Abra no Safari ou Chrome.

Instalação no iPhone:
1. Abra no Safari.
2. Compartilhar.
3. Adicionar à Tela de Início.

Instalação no Android:
1. Abra no Chrome.
2. Instalar app.
